# DevFix Hub

Practical frontend fixes for HTML, CSS, and JavaScript. Production-ready solutions for real-world developer problems.

## 🚀 Quick Start

1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/DevFix-Hub.git`
2. Open `index.html` in your browser
3. Browse solutions in [Blog](blog.html) or use [Tools](tools.html)

## 📁 Structure

- `index.html` - Landing page with featured solutions
- `blog.html` - All frontend fixes with search/filter
- `data/posts.json` - Blog content (add your own solutions here)
- `assets/` - CSS, JS for the site
- `create-post.html` - Add new blog posts

## ✨ Features

- Responsive design (mobile-first)
- Search across all solutions
- Category filtering (HTML/CSS/JS/Debugging)
- Dark mode toggle
- Smooth animations & particles background
- Production-ready code snippets

## 🛠️ Adding Content

1. Edit `data/posts.json` - add new post objects
2. Use `create-post.html` for guided editing
3. Site rebuilds automatically
